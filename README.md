# ChoreForge

ChoreForge is a rule-based household task management system designed to organize, assign, and track daily and weekly chores among multiple users.

The application introduces a structured approach to domestic task management by combining scheduling rules, task grouping, and a lightweight gamification system based on points, penalties, and collaboration bonuses.

## Features (Work in Progress)

- Task scheduling based on configurable rules
- Group-based task generation (e.g., room cleaning)
- Automatic task assignment among users
- Daily and weekly task planning
- Scoring system with rewards and penalties
- Collaboration bonus mechanics

## Project Status

This project is currently in early development.

The initial focus is on building a robust backend engine capable of generating tasks dynamically from scheduling rules.

## Tech Stack

- Java
- Spring Boot
- REST API (planned)
- MySQL (planned)

## Vision

The goal of ChoreForge is to transform household task management into a structured, fair, and engaging system, improving organization and collaboration in shared living environments.
